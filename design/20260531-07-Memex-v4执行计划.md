## 0. 约定

| 项目 | 说明 |

## 2. v0.0 Validation MVP — 详细 Sprint 计划

### Sprint 1：骨架 + Claude adapter + Storage（第 1-2 周）

**目标**：跑通「单 adapter 采集 → 写入 Markdown + SQLite → FTS5 可查」这条最小完整链路。

| # | 任务 | 类型 | 阻塞 | 预估 |
|---|---|---|---|---|
| 1.1 | 初始化 `memex` CLI 项目结构（pyproject.toml / Click / 目录布局） | 骨架 | — | 0.5d |
| 1.2 | 实现 `~/.memex/` 工作目录创建逻辑 | 骨架 | 1.1 | 0.5d |
| 1.3 | 实现 `config.toml` 默认配置生成 | 骨架 | 1.2 | 0.5d |
| 1.4 | 实现 `redactions.yaml` 默认脱敏规则生成 | 骨架 | 1.2 | 0.5d |
| 1.5 | 实现统一错误类型和 CLI JSON 输出格式 | 骨架 | 1.1 | 0.5d |
| 1.6 | 实现 Claude Code adapter（读取 `~/.claude/projects` JSONL） | 采集 | 1.5 | 2d |
| 1.7 | 实现按 mtime + offset 增量扫描 | 采集 | 1.6 | 1d |
| 1.8 | 实现 SQLite schema 初始化（sources/sessions/messages/chunks） | 存储 | 1.5 | 1d |
| 1.9 | 实现 `chunks_fts` 虚拟表 + INSERT trigger | 存储 | 1.8 | 0.5d |
| 1.10 | 实现 SQLite WAL 模式 | 存储 | 1.8 | 0.25d |
| 1.11 | 实现 Markdown session 文件写入 | 存储 | 1.5 | 1d |
| 1.12 | 实现「MD 先写 → SQLite 后写 → offset 最后更新」写入顺序 | 存储 | 1.8, 1.11 | 0.5d |

**关键路径**：1.1 → 1.5 → 1.6 → 1.7（增量采集证明可行）→ 1.12（写入顺序正确性）

**Sprint 1 验收标准**：
- `memex ingest` 可以从真实 Claude Code 历史读取 ≥ 10 条消息。
- Markdown 文件可人工打开阅读。
- `SELECT * FROM chunks_fts WHERE chunks_fts MATCH 'redis'` 有结果。

| 2.7 | 实现内置 redaction 规则（email / phone / API key / IP / 银行卡） | 处理 | — | 1d |

## 8. 关键风险与开放问题

| # | 风险/问题 | 影响 | 缓解策略 |
|---|---|---|---|
| R1 | Cursor `state.vscdb` 格式可能跨版本不稳定 | adapter 失效 | Sprint 2 spike 提前验证；做版本检测 fallback |
| R2 | macOS 访问 `~/.cursor/` 可能需要 Full Disk Access | 用户拒绝授权 | 提示引导；CLI fallback 手动 ingest |
| R3 | FTS5 中文分词质量（unicode61 对中文分词效果差） | 搜索召回低 | Sprint 3 spike jieba；考虑 simple tokenizer + bigram |
| R4 | Ollama 本地模型内存占用大，用户可能不安装 | 无摘要 | 降级方案已设计（pending_local_llm）；不阻塞入库 |
| R5 | rumps 在 macOS Sequoia 限制多 | menubar 无法渲染 | Sprint 9 预研；若不行直接跳 Tauri |
| R6 | 单人开发 v0.3 Rust 重写周期长 | 交付延期 | v0.2 Gate 严格执行；若 PMF 不够强不进入 v0.3 |
| R7 | SQLite WAL + 多进程并发写（daemon + CLI rebuild-index） | 数据损坏 | WAL + 单写入点；rebuild-index 需要 stop daemon |

### 开放决策（需在对应 Sprint 前确定）

| # | 问题 | 决策截止 | 选项 |
|---|---|---|---|
| D1 | 中文分词方案 | Sprint 3 开始前 | A) unicode61 B) jieba 外部分词 C) simple+bigram |
| D2 | Python CLI 框架 | Sprint 1 | A) Click B) Typer C) argparse |
| D3 | daemon HTTP 框架 | Sprint 7 开始前 | A) Flask B) aiohttp C) FastAPI |
| D4 | 是否跳过 v0.2 rumps 直接用 Tauri | Sprint 9 之前 | 取决于 rumps 在 macOS Sequoia 兼容性 |
| D5 | Rust async runtime | Sprint 10 开始前 | A) tokio B) async-std |
