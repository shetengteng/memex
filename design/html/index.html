TL;DR

7 tab → 5 一级菜单 + ⌘K 全局命令面板。 Overview 改成 Today（叙事 + 行动）， Sessions + Projects 合并为 Library（Tabs + Facets + Sheet drawer）， Reports + Reflect + Workload 合并为 Insights（3 个二级 tab）， IDE / MCP 从 Settings 提到一级菜单 Connect。 所有交互改用 shadcn-vue 的 Sidebar / Command / Sheet / Tabs / Skeleton 等 block 级旗舰组件。

变更对照

现状 新方案 理由
Overview tab (KPI 仪表盘) Today (叙事 + 行动) 回答"我今天干啥/能接着干啥"
Sessions tab + Projects tab Library (Tabs + Facets + Sheet) 同一份资料库，多维度切换 / 详情不跳页
Reports + Reflect + Workload Insights (3 个二级 tab) 都是"对历史的二次加工"
IDE 集成深埋 Settings Connect (一级菜单) MCP 是 Memex 核心价值，必须前置
Popup Search + Dashboard Search (两份实现) ⌘K 全局 Command Palette 统一搜索入口，符合生产力软件惯例
11 个基础 shadcn-vue 组件 +15 个组件 (含 P0/P1/P2) 用 Sidebar / Command / Sheet 等 block 级组件
