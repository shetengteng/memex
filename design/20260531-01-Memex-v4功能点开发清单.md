## 横向质量要求

### 隐私与安全

- [ ] 默认不上传任何会话内容到云端 LLM。
- [ ] Claude fallback 必须显式 opt-in。
- [ ] 云端调用前必须 redaction。
- [ ] 配置文件中不明文存储 API key。
- [ ] `credentials.toml` 权限必须为 600 或 700。
- [ ] HTTP server 仅监听 `127.0.0.1`。
- [ ] private session 默认不进入 MCP 搜索结果。

### 可观测性

- [ ] 每次搜索写入 `access_log`。
- [ ] 每日调用统计写入 `metrics`。
- [ ] 慢查询大于 500ms 记录 warning。
- [ ] adapter 解析失败记录错误计数。
- [ ] `memex doctor` 输出 adapter、SQLite、daemon、MCP、LLM 状态。

### 测试

- [ ] redaction 单元测试。
