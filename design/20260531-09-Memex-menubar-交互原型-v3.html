单 Popup 模型

所有功能聚焦在一个 360px popup 内。底部 3 个图标切换视图：

🔍 搜索（默认）— 输入即过滤，选中即跳转
⚙️ 设置 — adapter 开关 / LLM / 隐私 / 备份
📊 状态 — doctor 输出，一眼确认健康

交互流程

⌘⇧M → 弹出（默认搜索态）
输入 → 实时过滤
↑↓ / 鼠标 → 选中
⏎ → 跳转 IDE + 自动关闭
点击 ⚙️ → 切到设置（同 popup）
Esc → 关闭

为什么不需要独立窗口

API key — 设置视图显示状态，点击跳 CLI 命令提示
Adapter 开关 — 设置视图直接 toggle
Doctor — 状态视图一屏展示
Report — CLI 生成，menubar 只展示摘要

与 v1/v2 对比

v1: 6 tab mini app (440px, 593行)
v2: 2 tab + 右键下沉 (440px, 480行)
v3: 0 tab 单 popup (360px, 230行)
    底部 icon 切视图，不切 tab
