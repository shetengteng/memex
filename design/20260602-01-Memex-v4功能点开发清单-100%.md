## 13. 风险与开放决策

| # | 风险 | 缓解 |
|---|---|---|
| R1 | Cursor vscdb 格式跨版本不稳定 | agent-transcripts JSONL 方式已 bypass |
| R2 | macOS Full Disk Access 权限 | 引导提示 + CLI fallback |

| R4 | Ollama 内存占用大 | 已实现：LLM 失败不阻塞入库 |
| R5 | Rust 编译慢影响迭代速度 | cargo workspace + 增量编译 |

| # | 开放决策 | 状态 |
|---|---|---|
| D1 | 中文分词方案（jieba-rs / bigram） | 已决：bigram NEAR expansion |
| D2 | Tauri updater vs Sparkle | 待决 |
